# 👕 BoomDrops

Sistema que desarrollé para gestionar los drops de ropa vintage en mi tienda BoomCloset.

La idea nace porque necesitaba tener más claridad sobre qué estaba pasando en cada drop:
qué se vende más, cuánto realmente estoy ganando y qué tipo de ropa debería volver a comprar.

---

## 🚀 Qué hace el sistema

- Crear drops
- Agregar items con costo y precio
- Marcar prendas como vendidas
- Ver cuánto invertí vs cuánto vendí
- Calcular automáticamente:
  - Ganancia real
  - ROI
  - % vendido del drop
- Ver cuáles drops han sido mejores
- Analizar categorías y marcas que más se mueven

---

## 🧠 Por qué lo hice

Antes llevaba todo más desordenado y no tenía claridad real de números.

Con este sistema ahora puedo:
- entender qué tipo de ropa funciona mejor
- tomar decisiones más inteligentes al comprar
- ver claramente si un drop fue bueno o no

---
